About

This website was developed as the final project for my Bronze Duke of Edinburgh skills section.

It consists of over 2000 lines of code and utilises the following technologies:

  • Python - a high level programming language. The flask module is used in particular to create a web server.
  • HTML - the skeleton of the website - displaying headings, text, etc.
  • CSS - controls the appearance of the website and makes it look decent.
  • JavaScript - allows things to occur on the website. Makes the website dynamic.

All of the above play an important part in this website, and without any of them, the website would be horrible, and possibly not even creatable!
